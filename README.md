Descripción del programa:
Programa desarrollado en C++ que permite ingresar números enteros y clasificarlos automáticamente en diferentes estructuras de datos según su valor.

Los números pares positivos se almacenan en una cola, los impares positivos en una pila, los negativos en una lista simplemente enlazada y los ceros se rechazan.

También se guarda un historial en una lista doblemente enlazada y se construye un árbol BST con los números positivos.

Explicación de la lógica

El usuario ingresa una cantidad N de números.

Cada número es evaluado:

Par positivo - Cola
Impar positivo - Pila
Negativo - Lista simple
Cero → Rechazado

Luego el sistema permite mostrar estructuras, construir BST, buscar valores, eliminar negativos según condición y mostrar estadísticas.
